Como funciona

  1. Coletamos o preço de cada produto várias vezes por dia, direto na API do Mercado Livre.
  2. Depois de 14 dias de histórico, calculamos o preço mediano real daquele produto.
  3. Só quando o preço de hoje cai bem abaixo dessa mediana é que ele ganha o selo verificado.
